# Swiggy Sales Analysis Dashboard

This project analyzes Swiggy food delivery sales data using Excel.

Tools Used:
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

Key Insights:
- Total Sales: ₹53.01M
- Average Rating: 4.34
- Average Order Value: ₹268
- Top City: Bengaluru

Features:
- Monthly Sales Trend
- Daily Sales Trend
- Sales by Food Category
- Sales by State Map
- Top 5 Cities Analysis

Purpose:
To demonstrate data analysis and dashboard creation skills using Excel.
